# セットアップ完全ガイド

## 📋 必要な環境

- Node.js (v18以上推奨)
- npm または yarn
- Brave ブラウザ

## 🚀 インストール手順

### 1. プロジェクトのセットアップ

```bash
# プロジェクトディレクトリに移動
cd youtube-account-switcher

# 依存パッケージをインストール
npm install
```

### 2. アイコンの準備

`public/icons/` ディレクトリに以下のファイルを配置:
- icon16.png
- icon48.png
- icon128.png

詳細は `public/icons/ICON_GUIDE.md` を参照してください。

**開発中は仮のアイコンでもOK**

### 3. ビルド

```bash
npm run build
```

ビルドが成功すると `dist/` ディレクトリが作成されます。

### 4. Braveに拡張機能をインストール

1. Braveブラウザを開く
2. アドレスバーに `brave://extensions/` と入力
3. 右上の「デベロッパーモード」を **ON** にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. `dist` フォルダを選択

✅ 拡張機能がインストールされました!

## ⚙️ Googleアカウントの設定

拡張機能を使用する前に、複数のGoogleアカウントをBraveに追加する必要があります。

### 手順:

1. [YouTube](https://www.youtube.com) にアクセス
2. 右上のプロフィールアイコンをクリック
3. 「アカウントを追加」を選択
4. 最大4つのGoogleアカウントでログイン
5. すべてのアカウントでログイン状態を維持

### アカウントの対応:

| 目的 | アカウント番号 | 推奨用途 |
|------|---------------|---------|
| 🎮 ゲーム・エンタメ | アカウント1 (0) | ゲーム実況、エンタメ |
| 📚 学習・技術系 | アカウント2 (1) | プログラミング、チュートリアル |
| 🎵 音楽 | アカウント3 (2) | ミュージックビデオ、ライブ |
| 💼 仕事関連 | アカウント4 (3) | ビジネス、セミナー動画 |

**注意**: アカウント番号は0から始まります（内部的には0,1,2,3）

## 🎨 カスタマイズ

### アカウント名の変更

1. 拡張機能のアイコンをクリック
2. 各アカウントに分かりやすい名前を入力
3. 「設定を保存」をクリック

例:
- アカウント1: 趣味用Gmail
- アカウント2: 学習用Gmail
- アカウント3: 音楽専用
- アカウント4: 会社用Gmail

### デザインのカスタマイズ

`src/content/AccountSwitcher.jsx` の `purposes` 配列を編集:

```jsx
const purposes = [
  {
    id: 'gaming',
    emoji: '🎮',  // 絵文字を変更
    label: 'ゲーム・エンタメ',  // ラベルを変更
    accountIndex: 0,
    color: '#ff6b6b',  // 色を変更
    description: 'ゲーム実況、エンタメ動画'  // 説明を変更
  },
  // ...
];
```

変更後は再ビルドが必要:
```bash
npm run build
```

## 🔧 開発モード

開発中はホットリロードが使えます:

```bash
npm run dev
```

ファイルを変更すると自動的に再ビルドされます。
ブラウザで `brave://extensions/` から「再読み込み」をクリックして変更を反映してください。

## 📝 使い方

1. YouTubeにアクセス
2. 自動的にモーダルが表示される
3. 視聴目的を選択
4. 該当するGoogleアカウントに自動切り替え

## 🐛 トラブルシューティング

### モーダルが表示されない

- ブラウザのコンソール (F12) でエラーを確認
- 拡張機能が有効になっているか確認
- ページをリロード (Ctrl/Cmd + R)

### アカウントが切り替わらない

- YouTube にログインしているアカウント数を確認
- 各アカウントでログイン状態が維持されているか確認
- ブラウザのキャッシュをクリア

### ビルドエラー

```bash
# node_modules を削除して再インストール
rm -rf node_modules
npm install
npm run build
```

### スタイルが反映されない

SCSS モジュールが正しく読み込まれているか確認:
```bash
# sassパッケージの再インストール
npm install -D sass
npm run build
```

## 📦 本番環境向けビルド

公開用にビルドする場合:

```bash
# 本番ビルド
npm run build

# distフォルダをzipに圧縮
cd dist
zip -r ../youtube-account-switcher.zip .
```

## 🔄 更新方法

コードを変更した後:

1. `npm run build` でビルド
2. `brave://extensions/` で「再読み込み」をクリック
3. YouTubeをリロードして動作確認

## 💡 ヒント

- 開発中は `npm run dev` を使用
- 本番環境では `npm run build`
- アイコンは後から追加・変更可能
- カスタマイズ後は必ず再ビルドとブラウザの再読み込みを実行
